== Description ==

Shows the total number of users, posts, and comments on your wordpress blog. 
Comes with a admin dash widget and a site widget. 
Each can be activated or deactivated and the site widget has complete format 
options to allow you to format exacly how you need the widget to look.

Some Great Features:

*   Site widget.
*   Admin dash widget.
*   Site and admin widgets can be turned off if you dont want to use them.
*   Site widget has complete format options to allow you to format exacly how you need the widget to look. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `eh-wordpress-totals` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugins options page and activate the features you want to use
